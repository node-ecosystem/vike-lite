# Monorepo of `vike-lite`

Vike Lite (pronounced _/vaɪk laɪt/_) is a _light_ web framework inspired by [Vike](https://vike.dev).

### ✨ Features
~ 🪶 **Light** ~ Zero dependencies and only essential validations<br>
~ ⚖️ **Functionalities** ~ [See differences](packages/vike-lite/doc/differences/INDEX.md)<br>

### 📦 Packages
| Package | Description
| - | -
| [vike-lite](packages/vike-lite) | Core, inspired by [vike](https://github.com/vikejs/vike)
| [vike-lite-solid](packages/vike-lite-solid) | UI adapter for SolidJS, inspired by [vike-solid](https://github.com/vikejs/vike-solid)

### 🫶 Contribution
Want to contribute to the project? Clone the repository and install the dependencies:
```sh
git clone https://github.com/node-ecosystem/vike-lite.git
cd vike-lite
yarn
```
